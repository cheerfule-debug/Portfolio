Accessible Document Design

Assignment Context: Create a professionally formatted document following universal design principles to ensure accessibility for all users, including those with disabilities.

Accessibility Features Successfully Implemented:

  • Clear Heading Hierarchy: Logical H1, H2, H3 structure for screen readers
  • Readable Typography: Sans-serif font with appropriate sizing for easy reading
  • Organized Information: Bulleted lists and tables for easy scanning
  • Logical Reading Order: Information flows logically from overview to application process
  • Contact Information: Multiple ways to reach the organization (phone, email, mail)
  • Step-by-Step Instructions: Clear, numbered application process

Document Overview: "City of Tawny Creek Volunteer Program"

Document Purpose: Comprehensive guide for potential volunteers explaining opportunities, requirements, and application processes.

Key Sections:

  • Program Overview and Requirements
  • Volunteer Opportunities (Library, Museum, Parks & Recreation)
  • Sports Team Sponsorship Information (with organized contact tables)
  • Step-by-step Application Process
  • Complete Contact Information

Before: Original Coursework Document

Provided Course Material - Issues Identified:

  • ❌ Dense layout: Large blocks of text without sufficient white space
  • ❌ Image placement: Photos interrupted text flow and reading order
  • ❌ Complex tables: Contact information mixed within paragraph text
  • ❌ Visual clutter: Multiple design elements competed for attention
  • ❌ Inconsistent formatting: Varying text treatments throughout document

The original course material prioritized visual appeal over accessibility and usability.

After: My Accessible Redesign

Accessibility Improvements I Made:

  • ✅ Clean hierarchy: Created clear heading structure (H1 → H2 → H3)
  • ✅ Logical flow: Reorganized information in sequential, scannable order
  • ✅ Simplified tables: Restructured contact information by clear categories
  • ✅ Consistent formatting: Applied uniform spacing, fonts, and layout throughout
  • ✅ Text-focused design: Removed visual distractions for better readability
  • ✅ Step-by-step process: Organized application instructions into clear actions

My redesign prioritizes user experience and accessibility compliance.

Key Learning Demonstrated:

Assignment Process: This before/after comparison shows my ability to analyze existing course materials for accessibility issues and systematically improve them. I transformed provided content by simplifying complex layouts, removing visual distractions, and prioritizing clear information hierarchy over decorative elements.

Accessibility Analysis: Identified barriers in provided material and systematically addressed each issue
Usability Improvement: Redesigned content to be easier to scan and navigate for all users
Universal Design Application: Applied course principles to create a document that works better for everyone

Document Demonstrates Understanding Of:

Information Architecture: Logical organization from general info to specific application steps
User Experience: Multiple volunteer options accommodate different interests and time commitments
Accessibility Standards: Clear headings, readable fonts, and organized layout

Digital Publications

Assignment Context: Design professional flyers and posters that effectively communicate information while applying typography and layout principles.

Flyer Design

Communication Goals: Build local community around vintage car culture in South Pierce County, providing an alternative to larger commercial events. Target audience includes car enthusiasts, families, and general public of all demographics.

Design Elements Applied:

  • Visual hierarchy: Large title draws attention, followed by date/location, then details
  • Contrast: Light text on vintage car background ensures readability
  • Alignment: Left-aligned bullet points with center-aligned headers create organized layout
  • Typography: Bold, condensed sans-serif conveys energy and accessibility
  • Background choice: Authentic vintage car photo reinforces event theme

Poster Design

Format Adaptation: Same design content as flyer but optimized for poster format - larger scale for wall mounting and viewing from distance. Demonstrates understanding of how one design concept can be adapted for multiple publication formats.

Professional Design Practice: Creating multiple format versions from a single design concept is standard in the industry - allows for consistent branding across different media while optimizing readability and impact for each specific use case (handheld flyer vs. wall-mounted poster). Both created using ChatGPT.

Typography Principles Applied

Both publications demonstrate effective use of:

  • Typeface Categories: Bold condensed sans-serif fonts appropriate for event promotion
  • Contrast: Light text on darker background ensures readability
  • Hierarchy: Clear information hierarchy through font size and weight variation
  • Readability: Text sized appropriately for intended viewing distance (closer for flyer, farther for poster)

Storyboard & Video Production

Assignment Context: Plan and execute a video project using storyboarding techniques, then create the final video with proper accessibility features.

Storyboard Planning

Project: "How to Make an Avocado, Lettuce, and Tomato Sandwich"

Tool Used: PowerPoint (after testing multiple specialized storyboard tools)

Process Learning: Discovered the limitations of free storyboard software and adapted by using familiar presentation tools to achieve professional results.

Final Video

Accessibility Features:

  • Video hosted on YouTube with automatic caption generation available
  • Clear visual demonstration of each step

Storyboard to Video Relationship: The four-panel storyboard effectively guided the video production process, ensuring logical flow from introduction through ingredient presentation, preparation steps, and final result. This demonstrates planning skills and the ability to translate static planning documents into dynamic video content.
